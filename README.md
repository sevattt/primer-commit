Book Scraper
 Este script en Python scrapea libros de books.toscrape.com y guarda los títulos y precios en un Excel automáticamente.
Tecnologías
Python, requests, BeautifulSoup4, pandas
Cómo usarlo:

Clona el repositorio
Instala las dependencias con pip install -r requirements.txt 
Corre el script con python scraper.py
Se genera un archivo resultados.xlsx con todo
